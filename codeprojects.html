image gallery modal

A self-contained module of HTML, CSS, and JavaScript that can be added to a webpage to create a simple, responsive image gallery overlay on click. It's written without any additional libraries or frameworks.

"now playing" title export

This is a chrome extension that allows streamers to have the title of their music displayed as text in OBS. More specifically, the extension writes the title of an audible Youtube/Souncloud/etc tab to a plain .txt file. OBS (Open Broadcaster Software) can then read the dynamically updating file as a text source.
