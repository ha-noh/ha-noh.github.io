software projects

I develop tools for other artists and online content creators. Creatives with a do-it-yourself mentality will run into many shared problems in their workflow - my open-source solutions are intended to help ease that burden.

image gallery modal

A self-contained module of HTML, CSS, and JavaScript that can be added to a webpage to create a simple, responsive, and attractive image gallery overlay on click. It's written without any additional libraries or frameworks.

"now playing" title export

A chrome extension that allows streamers to have the title & artist for their music displayed as text in OBS. The extension writes the title of a Youtube/Souncloud/etc tab to a plain .txt file, which OBS (Open Broadcaster Software) can then read as a text source.
